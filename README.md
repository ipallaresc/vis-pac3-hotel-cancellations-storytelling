# La incertesa de les reserves

Web estàtica per a GitHub Pages amb visualitzacions de Flourish incrustades.

## Fitxers

- `index.html`: estructura de la pàgina i embeds de Flourish.
- `style.css`: estils visuals de la web.

## Publicació amb GitHub Pages

1. Crea un repositori a GitHub amb el nom `vis-pac3-hotel-cancellations-storytelling`.
2. Puja `index.html` i `style.css` a l'arrel del repositori.
3. Ves a **Settings → Pages**.
4. A **Build and deployment**, selecciona **Deploy from a branch**.
5. A **Branch**, selecciona `main` i carpeta `/root`.
6. Desa els canvis.
7. GitHub generarà una URL pública al cap d'uns minuts.

## Edició

Per canviar textos, edita directament les seccions de `index.html`.
Per canviar colors, edita les variables CSS dins `:root` a `style.css`.
